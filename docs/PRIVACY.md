# Privacy and Sanitization

This repo is intended to be safe to publish publicly.

Rules for tracked files:

- No real bulb suffixes or addresses
- No local filesystem paths
- No account names, tokens, or auth artifacts
- No raw captures that identify a personal environment

Examples in docs use generic placeholders like `ABCD`.

The local `vendor/` directory is research material only and is excluded from the public package and git history for this project.
